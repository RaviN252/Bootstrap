LISTS

types of list

order and un order list

for unorder list we use UL

  • apple
  • bannana
  • cucumber
  • dog

for order list we use OL

  1. apple
  2. bannana
  3. cucumber
  4. dog

as we can see the diffrent between order list and un order list

we can use list inside list either order un order

  • apple
    • Kashmir apple
    • pine apple
    • ooty apple
  • apple
    1. Kashmir apple
    2. pine apple
    3. ooty apple
  • bannana
  • cucumber
  • dog
